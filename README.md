# asm-clickdebounce


使用ASM对Android的点击事件进行拦截，防止点击抖动的发生。

我写了博客[《拦截控件点击-巧用ASM处理防抖》](https://www.jianshu.com/p/28751130c038)，分享我对点击抖动的思考和解决思路。
